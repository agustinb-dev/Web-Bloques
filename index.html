Articulos donde se explica la diferencia entre los elementos contenedores HTML. Hay dos tipos principales de estos elementos: los elementos en bloque y los elementos en linea, en esta web vamos a profundizar sobre cuales son las funciones de estos elementos y daremos algunos ejemplos de los mismos.

Elementos HTML en bloque:

Los elementos en bloque en html son los que los navegadores toman con el valor css "display: block;" por default, esto significa que comienzan en una nueva linea y abarcan todo el ancho que halla disponible.

Ejemplo comunes de elementos de bloque son:

Elemento: <p>: Este elemento define un bloque de texto que comienza en una nueva linea. Ejemplo:

Elemento <div>: Este elemento define un bloque contenedor que comienza en una nueva linea.

Elemnto <ul>: Este elemento crea un nuevo bloque que contiene una lista desordenada.

Elementos HTML en linea:

Los elementos en linea en html son los que los navegadores toman con el valor css "display: inline;" por default, esto significa que no crean una nueva linea y solo ocupan el ancho necesario.

Ejemplo comunes de estos elementos son:

Elemento: <a>: Este elemento define enlace un de hypertexto en la misma linea.

Elemento <img>: Este elemento define una imagen en la misma linea.

Elemento <span>: El elemento <span> es la version en linea del elemento <div>
